# Graphic Editor (Java)

Java Swing 기반으로 구현한 간단한 그래픽 에디터 프로젝트입니다.

## Features
- 사각형 그리기
- 도형 선택 및 이동
- 메뉴바 및 툴바 구성

## Tech Stack
- Java
- Swing (GUI)

## Structure
- `GMainFrame` : 전체 프레임 구성
- `GDrawingPanel` : 도형 그리기 영역
- `GMenuBar` : 메뉴 구성
- `GToolBar` : 도구 선택 기능

## How to Run
1. `Main.java` 실행
2. 그래픽 에디터 창 실행
